# Week 1

## MATH BODIES
### Gruppe: + -
- Neutral element 0
### Ring: + - *
- A(B+C) = (AB) + (BC)
- Neutral Element usually 1
- Have an inverse $\pm$ element -a
- Have an inverse multiplication element of $\frac{1}{a}$
### Body: + - * /
- Neutral Element usually 1

Example body:
GF(5) means that the body contains {0,1,2,3,4} otherwise mod 5
